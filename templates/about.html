About This Todo App

This is a simple Todo application built using Flask and SQLite for database management. You can add, update, and delete your tasks easily.

Features:
  • Add your tasks with a title and description.
  • View all your tasks in a table format.
  • Update any task to keep it up-to-date.
  • Delete tasks that are completed or no longer needed.
  • Responsive design using Bootstrap 5.
How to Use:
  1. Go to the home page and add your tasks.
  2. Click "Update" to modify any task.
  3. Click "Delete" to remove a task.
  4. All changes are saved automatically in the database.

Built By Ansh using Flask, SQLAlchemy, and Bootstrap.
